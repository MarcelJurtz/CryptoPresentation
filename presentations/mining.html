Mining

by example Bitcoin
  • Most people think of mining as the way to create new bitcoins
  • That is not wrong, but only a side product of the process

Creating new blocks

  • New blocks are created by calculating hash values

Creating new blocks

  • The hash value is valid, if it has a specific amount of leading zeroes

Increasing difficulty

  • Difficulty is adjusted each todo blocks
  • More mining power leads to higher difficulty
  • Which leads to higher mining power...
  • So the reward will be only around the energy/device prices
  • You won't get rich by mining, only getting back what you first put in (plus maybe minor extra)
